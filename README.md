# Assignment_Task0

This is a simple Spring Boot project that demonstrates basic CRUD operations using H2 Database and Thymeleaf for the frontend.

## Features

- Add a new user
- View all users
- Delete a user

## Technologies Used

- Spring Boot
- Spring Data JPA
- H2 Database
- Thymeleaf
